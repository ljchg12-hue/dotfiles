# /ui - Instant UI Component Generator

Generate production-ready UI components with optimal tool selection.

## Usage

```bash
/ui [description]
```

## Examples

```bash
/ui dashboard with 5 sector cards and trend charts
/ui responsive table with sorting and filtering
/ui animated timeline component
/ui form with validation for user search
/ui dark mode card grid with glassmorphism
```

## What it does

1. Analyzes your UI requirements
2. Selects optimal React libraries (Tremor/shadcn/Recharts)
3. Generates complete component code
4. Provides installation commands if needed
5. Includes usage examples and documentation

## Automatic Tool Selection

- **Dashboard/Metrics** → Tremor
- **Forms/Tables** → shadcn/ui
- **Charts** → Recharts or Observable Plot
- **Animations** → Framer Motion

## Output

- Complete React component code
- TypeScript interfaces
- TailwindCSS styling
- Sample data
- Installation commands
- Usage examples

## Global Activation

This command works in ANY project, ANY directory.
No setup required.
